# Go "Git" Em!
A git reimplimentation using golang

## Goal
The goal was to create a simple implimentation of the core concept of git.

## Features
- init 
  - Initialize repo in current dir
- add 
  - Adds files to the staging area 
- status 
  - Shows the staging area and current status
- commit 
  - Generates a new commit. This includes:
    - hash
    - user 
    - email
    - message
- push
  - Updates remote commit tree to add the new commit
